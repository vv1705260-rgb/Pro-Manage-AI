# 🚀 ProManage AI

A modern full-stack **Project Management Tool** with real-time collaboration, built using MERN stack.

---

## ✨ Features

* 🔐 JWT Authentication (Login / Register)
* 📊 Drag & Drop Kanban Board
* ⚡ Real-Time Task Updates (Socket.io)
* 📌 Task Management (Create, Update, Delete)
* 🎨 Modern UI with Glassmorphism Design
* 📱 Responsive Layout

---

## 🛠 Tech Stack

**Frontend:**

* React.js
* Tailwind CSS

**Backend:**

* Node.js
* Express.js

**Database:**

* MongoDB

**Real-Time:**

* Socket.io

## ⚙️ Installation & Setup

### 1️⃣ Clone Repository

```bash
git clone https://github.com/vv1705260-rgb/Pro-Manage-AI.git
cd Pro-Manage-AI
```

### 2️⃣ Install Dependencies

```bash
npm install
cd client && npm install
cd ../server && npm install
```

### 3️⃣ Run Project

```bash
npm run dev
```
